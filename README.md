## Project Overview
This project focuses on modernizing the Foot Locker mobile experience by merging "Sneaker Culture" aesthetics with high-performance e-commerce strategies. The goal was to build a seamless user journey, from product discovery to final checkout, while maximizing business KPIs.

## Key features & Business impact
1. Immersive sneaker branding (UI/UX)
Objective: Strengthen brand identity for the sneakerhead community.

Solution: A visual overhaul using bold typography, premium product photography, and a modern interface.

2. User engagement & Retention (Auth)
Objective: Convert anonymous visitors into identified, loyal customers.

Solution: Seamless login/signup flow to enable order tracking and collect valuable first-party data.

3. Checkout optimization (Conversion)
Objective: Drastically reduce cart abandonment rates.

Solution: Integration of multiple payment gateways (digital wallets, credit cards) to match global user habits and ensure a secure transaction.

4. Smart cross-selling (AOV - Average Order Value)
Objective: Increase the average basket size.

Solution: Implementation of a "Frequently Bought Together" module (cleaning kits, socks, accessories) directly within the checkout funnel.

5. Enhanced customer support
Objective: Minimize post-purchase friction and improve trust.

Solution: Intuitive contact form and FAQ access designed for quick mobile interaction.

## Tech stack & Skills
Design: Figma (Components, Auto-layout, Advanced Prototyping).

Product Management: Functional specifications, User Flow mapping.

Analysis: Conversion Rate Optimization (CRO) principles & A/B Testing logic.
